Installation:

  1. Download the plugin from SpigotMC.org
  2. Place the plugin into the plugins folder on your server. By default this is located at '<Server Directory>/plugins/'
  3. Turn on the server, once it is started turn it off again. This is to generate the config files
  4. Configure the issue detection to be adjusted to your liking. If you don't know what you are doing here, the defaults are fine.
  5. Configure the actions to take in response to each issue. By default only 'The Archive' (Logging issues) is enabled. You will need to setup the other actions to utilise them.
  6. Read Server Assistant's commands and permissions page and assign the relevant permissions in your Permission Manager plugin (If you have one, if you do not then /op will be required to perform commands)
  7. Turn on the server and use '/crashserver' and '/lagserver' to test that everything is working
  8. Relax! Enjoy Server Assistant watching and looking after your server!
